# About the game
Urban Stories o Historias Urbanas es una novela interactiva situada en México en los años 80/90 en pleno boom del punk y rock urbano. Nuestro protagonista es un policía investigando el caso de un "Asesino de punks". Para hallarlo, deberá buscar pistas en diferentes calles de la ciudad. Este juego está inspirado en juegos como Snatcher con una línea gráfica pixel art.
